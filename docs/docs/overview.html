Introduction to RFDT - A Physically Based Differentiable RF Digital Twin Framework

Core Components

RFDT consists of two main components that work together to provide a complete differentiable RF simulation experience:

PyRFDT

PyRFDT is RFDT's core computational module that can run independently. It provides the fundamental differentiable ray tracing algorithms, material modeling, and simulation engine capabilities.

RFDT Editor

RFDT Editor is a web-based visual interface that requires connection to a local or remote server hosting PyRFDT. It provides intuitive scene creation, real-time visualization, and interactive simulation capabilities.

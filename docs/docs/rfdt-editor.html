RFDT Editor

Web-based visual interface for RF design and simulation

What is RFDT Editor?

RFDT Editor is a web-based visual interface that requires connection to a local or remote server hosting PyRFDT. It provides intuitive scene creation, real-time visualization, and interactive simulation capabilities for RF system design and optimization.

Design Tools

  • • 3D scene editor
  • • Drag-and-drop interface
  • • Material property editor
  • • Antenna placement tools

Visualization

  • • Real-time ray visualization
  • • Heat map overlays
  • • Signal strength maps
  • • Interactive charts

Getting Started

1

Prerequisites

Ensure you have PyRFDT installed and running before launching the RFDT Editor.

Note: The RFDT Editor requires an active PyRFDT server connection for simulations

2

Launch Editor

Access the RFDT Editor through your browser. No installation needed!
